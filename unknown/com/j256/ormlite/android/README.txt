BACKGROUND:

This package provides a lightweight Object Relational Mapping between Java classes and the Android
SQLite database calls.  There is also a JDBC version of the package which works with many JDBC
databases under other operating systems.

For more information, see the online documentation on the home page:

http://ormlite.sourceforge.net/
